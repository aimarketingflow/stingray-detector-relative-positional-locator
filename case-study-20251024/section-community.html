6.2 How to Submit Your Report

Submit to Community Reports:

  1. Fork the GitHub repository
  2. Add your report to community-reports/ folder
  3. Include:
    • Date and time of detection
    • Location (city/state - can redact exact address)
    • Frequency detected
    • Signal strength data
    • Photos (optional)
    • CSV scan files
  4. Submit pull request

6.3 Report to Authorities

üö® Official Reporting

FCC (Federal Communications Commission):

  • Website: consumercomplaints.fcc.gov
  • Phone: 1-888-CALL-FCC (1-888-225-5322)
  • Include: Frequency, signal strength, location, dates

FBI (Cyber Division):

  • Website: ic3.gov (Internet Crime Complaint Center)
  • Local FBI field office

Local Law Enforcement:

  • File report with local police
  • Provide all documentation and evidence
